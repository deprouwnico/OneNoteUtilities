# Get-ONHierarchy

## SYNOPSIS
Gets the current OneNote Hierarchy

## SYNTAX

```
Get-ONHierarchy [<CommonParameters>]
```

## DESCRIPTION
Loads the current OneNote Hierarchy for use by other functions

## EXAMPLES

### EXAMPLE 1
```
Get-ONHierarchy
```
